# 実行方法
1. テキストの各トークンの尤度を計算し、`likelihood`ディレクトリに保存する。（GPU推奨）
```

```

2. 保存された尤度を用いてMin-k% Probを計算し、AUCスコアを標準出力に表示する。
```
python src/eval.py
```
